DOM（Document Object Model）即文档对象模型，是 HTML 和 XML 文档的编程接口。

其中 HTML DOM 定义了所有 HTML 元素的对象和属性，以及访问它们的方法。换言之，HTML DOM 是关于如何获取、修改、添加或删除 HTML 元素的标准。

DOM 以树结构表达 HTML 文档。
