| **编号** | **依赖库版本**      | **核心类 (Gadget)**                           | **说明**                                      |
| ------ | -------------- | ------------------------------------------ | ------------------------------------------- |
| CC1    | CC 3.1 - 3.2.1 | `InvokerTransformer`                       | **经典款**。分为 TransformedMap 和 LazyMap 两个版本。   |
| CC2    | CC 4.0         | `PriorityQueue` + `TransformingComparator` | 针对 CC4 的链，利用了队列排序时的比较逻辑。                    |
| CC3    | CC 3.1 - 3.2.1 | `InstantiateTransformer`                   | CC1 的变体。不直接用 Runtime，而是通过加载字节码规避某些限制。       |
| CC4    | CC 4.0         | `PriorityQueue` + `InstantiateTransformer` | CC2 的变体。结合了 CC2 的入口和 CC3 的字节码执行方式。          |
| CC5    | CC 3.1 - 3.2.1 | `BadAttributeValueExpException`            | LazyMap 变体。利用了 Tiedo 异常类的 `readObject` 来触发。 |
| CC6    | CC 3.1 - 3.2.1 | `HashSet` / `HashMap`                      | LazyMap 变体。利用 Map 的 `hashCode` 触发，是最通用的链之一。 |
| CC7    | CC 3.1 - 3.2.1 | `Hashtable`                                | LazyMap 变体。利用 `Hashtable` 的 `equals` 逻辑触发。  |
