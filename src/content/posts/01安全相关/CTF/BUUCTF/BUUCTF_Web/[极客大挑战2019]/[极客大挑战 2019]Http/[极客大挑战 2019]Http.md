# [极客大挑战 2019]Http

打开链接一个简单的页面，什么也看不出来。查看源码，发现下方有`Secret.php`文件。直接 URL 访问。

![[attachments/Pasted image 20240929221216.png]]

页面显示 `It doesn't come from 'https://Sycsecret.buuoj.cn'`，显然是 Refer 惹的祸。（题目都叫 Http 了，也不考虑还有别的什么东西，直接干。）
直接用浏览器插件 ModHeader - Modify HTTP headers 对请求头进行修改，刷新。（Burpsuite 抓包也可以搞）
![[attachments/Pasted image 20240929221433.png]]

页面显示`Please use "Syclover" browser`，browser 包 User-Agent 的。同样改一下，刷新。

![[attachments/Pasted image 20240929221551.png]]

再次提示`No!!! you can only read this locally!!!`，locally 意思是需要本地访问啊，再修改刷新。

![[attachments/Pasted image 20240929221639.png]]

得到`flag{2ae85e1e-ea3b-44e7-869d-cf1e6360526d}`。

#### 补充

Refer，页面（链接）跳转时 Refer 用来表示跳转来源。
User-Agent，标注请求发送方设备、浏览器等信息。
X-Forward-For，HTTP 请求端的源 IP ，包括起始 IP 和路径上经过的代理 IP ；只有在通过了 HTTP 代理或者负载均衡服务器时才会添加该项。
