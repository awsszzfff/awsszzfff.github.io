## 域名（Domain Name）

DNS（Domain Name System）域名解析系统，域名和 IP 地址之间的映射关系。

| 顶级域名 | 二级域名      | 三级域名          |
| ---- | --------- | ------------- |
| .com | baidu.com | www.baidu.com |

| 政府域名 | 商业域名 | 教育域名 |
| ---- | ---- | ---- |
| .gov | .com | .edu |

二级域名是指顶级域名之下的域名，在国际顶级域名下，它是指域名注册人的网上名称，例如 ibm，yahoo， microsoft 等；在国家顶级域名下，它是表示注册企业类别的符号，例如 com，top，edu，gov，net 等；

## HTTP 请求头

XFF（X-Forward-For）：用于记录客户端的真实 IP 地址和请求经过的代理 IP；

Referer：用于记录发起请求的来源页面 URL；
