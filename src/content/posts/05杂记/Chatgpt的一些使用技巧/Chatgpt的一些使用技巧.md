# Chatgpt的一些使用技巧

```Prompt
结构化提示词
定义角色：你是XXX
定义工作流：workflow
eg：
1. 生成课题
2. 生成XXX的大纲
3. 根据大纲的每个章节生成具体的XXX内容 
```

# 论文阅读

```Prompt
详细描述本篇文章的主要内容，创新点，具体方案方法等
详细描述3.2节的内容，包括其中的公式，符号含义等
重新对第3.2节进行详细完善的描述，包括对其中的公式和符号的解释【/完善刚在的描述】
对公式1及其相关的上下文进行详细的解释
详细描述4.2节的内容
解释一下表1
解释一下图1
```
