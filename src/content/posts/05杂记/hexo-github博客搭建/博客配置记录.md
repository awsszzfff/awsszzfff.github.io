在 hexo 的根目录创建一个文件 `_config.butterfly.yml`，并把主题目录的 `_config.yml` 内容复制到 `_config.butterfly.yml` ；之后修改主题相关内容在该文件中修改。（这样主题文件夹中的`_config.yml`文件就不起作用了，方便之后对主题的升级）

## `_config.yml`

- theme: butterfly

- url: https://xxxxx.github.io/
- 
post_asset_folder: true

- deploy:
	type: 'git'
	repository: git@github.com:xxxxx/xxxxx.github.io.git
	branch: main
	message: "Auto-deployed by Hexo"

## `_config.butterfly.yml`
- nav

```txt
menu: # 目录
  首页: / || fas fa-home
  归档: /archives/ || fas fa-archive
  标签: /tags/ || fas fa-tags
  分类: /categories/ || fas fa-folder-open
  # List||fas fa-list:
  #   Music: /music/ || fas fa-music
  #   Movie: /movies/ || fas fa-video
  友链: /link/ || fas fa-link
  关于: /about/ || fas fa-heart
```

lightbox: fancybox  # 图片大图模式

- math
	- use: mathjax
- 404 page

## `source/`

tags 标签页

categories 分类页

link 友情链接

`_data` 友链页内容
